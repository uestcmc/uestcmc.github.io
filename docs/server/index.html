社区公益服务器介绍

生存服务器​

原版生电技术社区：专注红石自动化与极限资源管理，已建成全物品、集成树场、猪人塔塔等基础设施。适合追求工程美学与效率至上的硬核玩家。

USB PVP 大赛：服务器内还有一些建筑和像素画，其中有一座竞技场用于举办 “USB 杯” PVP 大赛，同全校的 PVP 高手同台竞技。

创造服务器​

校园数字方舟计划：UESTC 校园复原计划，采用小木斧批量建造、动态光影还原真实气象，主楼群复刻完成。成果将用于元宇宙校庆，招募擅长建筑、建模的“数字匠人”。

同时还作为生存服的实验基地，有一些生电装置、像素画、建筑小镇等。

【整合包】机械动力:齿轮盛宴 - Create Delight​

可下载 群文件/整合包资源/[Client]Create-Delight-Remake-v0.4.6.6-lite，拖入PCL2自动安装。

整合包介绍 机械动力：齿轮盛宴 将各种 机械动力、AE2、农夫乐事、Ad_Astra、Alex洞穴深度融合，创造出一个值得探索与建造的奇妙世界！ 相关视频链接 https://www.bilibili.com/video/BV1zvNozWECh/

【短期活动】国庆 生存派对​

10月1日迎来了我们的国庆

找不到想玩的服务器怎么办

部分高校的mc爱好者组织起来

搭建国庆专属“生存派对”服务器

每个学校都可以作为一个单元进入同一个服务器

介于参与人数可能会不足的问题 我们则与西电合作成为一个整体参加活动

服务器旨在后勤保障和插件保障过后的生存模式下搭建各种小游戏场景进行游玩

不同学校的小游戏设施可以互相体验

我们和西电选择了最基础的跑酷

此活动群成员自愿参加

有意向者可以扫描下方二维码进入大群了解更多

【整合包服】 BetterMC 整合包服务器​

整合包版本为 BMC 4 （Forge） v 40，可在 PCL 搜索 BMC 4 下载后，或下载群文件/整合包资源/Better MC [FORGE] 1.20.1 v 40 后拖入 PCL 自动安装
